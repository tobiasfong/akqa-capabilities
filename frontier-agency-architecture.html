Start with Option A (Industry RAG) for immediate deployment. Invest in Option C (RAG + LoRA Adapter) for verticals with deep client concentration. Reserve Option B (Fine-Tuned LLM) for verticals where AKQA intends to build defensible, proprietary advantage over time.
